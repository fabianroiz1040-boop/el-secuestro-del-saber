# El secuestro del saber

Repositorio fundacional de la obra **El secuestro del saber** y del marco epistemológico, jurídico y operativo de la **Fundación FHIA**.

Este repositorio documenta un sistema diseñado para impedir el secuestro del conocimiento, garantizar su trazabilidad ética y habilitar mecanismos de redistribución orientados a la dignidad humana.

---

## Marco FHIA

FHIA es una infraestructura ética de conocimiento, evaluación y redistribución.  
No constituye una ONG tradicional ni un organismo financiero.

Incluye:
- Documento Fundacional
- Protocolo Epistemológico
- Algoritmo H
- Pilares de la Dignidad Humana
- Marco de redistribución asistida
